# Прогнозирование спроса на прокат велосипедов

## Тема проекта

Прогнозирование спроса на прокат велосипедов на основе погодных и календарных данных.

## Цель проекта

Разработать модель машинного обучения для точного предсказания количества арендованных велосипедов в определенный час. Результаты работы модели могут быть использованы каршеринговыми сервисами для предиктивного распределения велосипедов по станциям во избежание их дефицита в часы пик.

## Источник данных

*Датасет: Bike Sharing Dataset.*
